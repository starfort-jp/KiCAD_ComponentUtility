1. What is it?
 "KiCAD_ComponentUtility" is a simple convertor. (Run under Windows 7 64bit / Windows 10 64bit.)
 You can convert a KiCad lib file into a CSV file or a CSV file into a KiCad lib file. 

2. Licence
 Please refer to the attached document "Licence-E.txt".

3. Usage
 1) Click "OPEN" for loading a KiCAD lib file. 
 2) Click "-> CSV" for change to CSV format.
 3) Click "SAVE" for saving a csv file.
 4) Edit a csv file. Then save it.
 5) Click "OPEN" for loading a edited csv file. 
 6) Click "-> LIB" for change to KiCAD lib format.
 7) Click "SAVE" for saving a KiCAD lib file.
note :  you can't edit text directly on this program.

4. Limitations
 1) Not support Unicode.
 2) Can't use a pin name prefix '+' or '-'.
