１．はじめに
　このソフトは、KiCad Schematic Libraries File と CSV File の相互変換を行うものです。
　変換後の CSV File を表計算ソフトで編集し、再度 KiCad Schematic Libraries File へ変換して使用します。

３．使い方
　１）メインフォームでのボタン操作
　　開く：ファイルを開いて内容をロードして表示します。
　　保存：ロードされている内容をファイルへ保存します。
　　CSVへ変換：表示されているテキストをCSV形式へ変換して表示します。
        LIBへ変換：表示されているテキストをKiCad LIB形式へ変換して表示します。
　　消去：表示されているテキストを消去します。
　　オプション："オプションの設定"フォームを開きます。
　　図形表示："グラフィック表示"フォームを開きます。
　２）基本的な使い方
　　1)開く ボタンを押し、KiCad の lib ファイルをロードします。
　　2)保存 ボタンを押し、csv 形式でファイルをセーブします。
　　3)表計算ソフトで編集し、csv 形式でセーブします。
　　4)開く ボタンを押し、セーブした csv ファイルをロードします。 
　　5)保存 ボタンを押し、lib形式でファイルをセーブして KiCad のライブラリへ取り込みます。
　備考：詳細な使い方については、同梱のマニュアルをご参照ください。

４．制限事項
　１）端子名に日本語を使った場合は文字化けします。
　２）端子名の先頭に算術記号(+,-)があると、Excel等で正常に読み込めません。
　　　これは表計算ソフトが数式と区別できないためです。
　　　ダブルクォーテーションで囲むと回避できますが、KiCad ファイルフォーマットとの整合性のため付加しておりません。
　３）KiCad のファイルフォーマットにデリミタがないため、列が乱れることがあります。
　　　再変換後、余分な空白は削除されます。
